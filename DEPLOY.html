Deployment Guide

Deploy InvoiceIQ Online

Deploy your backend to Railway and frontend to Vercel. Both are free to start. Total time: ~15 minutes.

⏱ ~15 minutes · Free tier available on both platforms
0
Before You Start
Install these if you haven't already
Required accounts & tools
  1. Create a free account at railway.app
  2. Create a free account at vercel.com
  3. Create a free account at github.com (needed to deploy)
  4. Install Git on your computer: git-scm.com
  5. Get your Anthropic API key from console.anthropic.com
1
Push Code to GitHub
Both Railway and Vercel deploy from GitHub
Terminal commands
# 1. Go to github.com → New Repository → name it "invoiceiq"
# 2. Then run these commands inside your invoiceiq folder:

cd invoiceiq
git init
git add .
git commit -m "Initial InvoiceIQ MVP"
git remote add origin https://github.com/YOUR_USERNAME/invoiceiq.git
git push -u origin main
2
Deploy Backend to Railway
Railway hosts your Node.js + Claude API server
Railway setup steps
  1. Go to railway.app → Click New Project
  2. Choose Deploy from GitHub repo → select your invoiceiq repo
  3. When asked for root directory, set it to: backend
  4. Click Deploy — Railway will auto-detect Node.js
  5. Once deployed, go to Settings → Variables and add:

VariableValueNotes
ANTHROPIC_API_KEYsk-ant-...Your Claude API key
FRONTEND_URLhttps://your-app.vercel.appAdd after Vercel deploy
PORT4000Railway sets this automatically

  1. Go to Settings → Networking → click Generate Domain
  2. Copy your Railway URL — it looks like: invoiceiq-backend.up.railway.app
✓ Test it: Open your Railway URL + /health in browser. You should see: {"status":"ok"}
3
Deploy Frontend to Vercel
Vercel hosts your React app with a global CDN
Vercel setup steps
  1. Go to vercel.com → Click Add New Project
  2. Import your invoiceiq GitHub repo
  3. Set Root Directory to: frontend
  4. Framework preset will auto-detect as Vite
  5. Under Environment Variables, add:

VariableValue
VITE_API_URLhttps://your-app.up.railway.app

  1. Click Deploy — Vercel builds and deploys in ~60 seconds
  2. Copy your Vercel URL — like: invoiceiq.vercel.app
4
Connect Frontend ↔ Backend
Update Railway with your Vercel URL to allow requests
Final step
  1. Go back to Railway → your project → Variables
  2. Update FRONTEND_URL to your actual Vercel URL:
    e.g. https://invoiceiq.vercel.app
  3. Railway will auto-redeploy with the new variable
  4. Open your Vercel URL — your app is live! 🎉
